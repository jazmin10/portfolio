CODEHIVE

Codehive is a MVC framework application designed to be a study guide and career development tool for new comers to the web development world. With Codehive, fellow developers can share their best interview tips, advice, and coding challenges.

CONCEPT

From my experience as a new comer to the web development world, learning new programming languages, as well as preparing for the interview process, can be intimidating. Slowly but surely you may feel overwhelmed and discouraged. However, what if you didn't have to experience this daunting task on your own? Hence, Codehive.

Codehive's purpose is to build a community of developers who can support and guide each other by creating an ongoing collection of programming knowledge.

For details, please view Code Hive's GitHub repository. To view an account in Code Hive, you may use demo for username and password.
